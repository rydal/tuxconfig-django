Tuxconfig device installer..

Module installs, they're a pain. You have to find your device id, get the chipset name. clone some source code. compile it into machine code, add in into your kernel with DKMS.

Simple eh?

Can we automate this process? Can we have a system which searches for your device online, and installs the appropriate software for it?

That's what this system is all about!


I will build a debian package for this after some submissions, until then I'm happy to merge pull requests!
